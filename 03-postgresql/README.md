# 03 - PostgreSQL Fundamentals

Use these files in pgAdmin, psql, or your SQL tool of choice.

## Competencies

- Schema and constraints
- One-to-many relationships
- Joins and aggregations
- Filtering, sorting, grouping
- Index basics

## Files

- `schema.sql`: Create tables and seed data.
- `queries.sql`: Practice query exercises.

## Tasks

1. Run `schema.sql`.
2. Complete each TODO in `queries.sql`.
3. Add one additional index and measure query impact with `EXPLAIN ANALYZE`.

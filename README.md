# Coding-Fundamentals-Practice

Practice repo for TypeScript, React, API routes, PostgreSQL, and MongoDB with AI-guided coaching.

## How to Use This Repo

1. Start with a module below.
2. Read its `README.md` and complete tasks in order.
3. Ask Copilot for hints, not full solutions first.
4. Run and test as you go.

## Learning Modules

- `01-typescript`: Types, unions, narrowing, generics, utility types, async patterns.
- `02-api-routes`: Express route design, validation, status codes, error handling.
- `03-postgresql`: Relational modeling, joins, aggregations, constraints, indexing basics.
- `04-mongodb`: Document modeling, CRUD, updates, aggregation pipeline basics.
- `05-react`: Components, props/state, list rendering, controlled forms, API integration shape.

## Teaching Style (Configured)

This repo includes `.github/copilot-instructions.md` so Copilot behaves like a coach:

- Gives small hints first.
- Points out likely mistakes and why they happen.
- Asks guiding questions when you are close.
- Provides complete solutions only when you ask.

## Suggested Weekly Flow

- Day 1: TypeScript
- Day 2: API routes
- Day 3: PostgreSQL
- Day 4: MongoDB
- Day 5: React + integration mini-project

## Final Challenge

Build a tiny full-stack app:

- React client for task tracking.
- API routes for CRUD operations.
- PostgreSQL for relational task/user data.
- MongoDB for activity/event logs.

Use this prompt to begin:

"Guide me with hints only through the final challenge. Ask me one step at a time."
